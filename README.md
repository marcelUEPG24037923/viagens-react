# Sistema de Gerenciamento de Viagens — React + Express

Projeto desenvolvido para a disciplina **Desenvolvimento Web**, focado na criação de uma interface completa utilizando **React (Vite)**, integrada a uma **API REST** construída em **Express.js**.  

O sistema permite listar, cadastrar, editar e excluir viagens, contendo os campos:

- **Destino**
- **Duração**
- **Valor**

## Tecnologias Utilizadas

### **Frontend (React + Vite)**
- React Hooks (useState, useEffect)
- Componentização
- Comunicação com API via Fetch
- Modal para formulários
- CSS modularizado

### **Backend (Node + Express)**
- API REST completa (CRUD)
- Controllers e rotas organizadas
- Middleware de CORS
- Integração com banco de dados

## Como executar o projeto

### 1. Clone o repositório
```bash
git clone https://github.com/SEU-USUARIO/SEU-REPO.git

### 2. Acesse a pasta do frontend
cd frontend

### 3. Instale as dependências
npm install

### 4. Execute o projeto
npm run dev

### 5. Acesse a pasta da API
cd api

### 6. Instale as dependências
npm install

### 7. Inicie o servidor
npm start

A API rodará em: http://localhost:3000

O frontend (Vite) rodará em: http://localhost:5173
