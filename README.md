Sistema de Gerenciamento de Viagens — React + Express

Projeto desenvolvido para a disciplina Desenvolvimento Web, com foco na criação de uma interface completa utilizando React (Vite), integrada a uma API REST construída em Express.js.

O sistema permite listar, cadastrar, editar e excluir viagens, contendo os campos:

Destino

Duração

Valor

Como executar o projeto:

1. Clone o repositório
git clone https://github.com/SEU-USUARIO/SEU-REPO.git

Frontend (React)
2. Acesse a pasta do frontend
cd frontend

3. Instale as dependências
npm install

4. Execute o projeto
npm run dev

Backend (Express)
5. Acesse a pasta da API
cd api

6. Instale as dependências
npm install

7. Inicie o servidor
npm start

A API rodará em: http://localhost:3000

O frontend (Vite) rodará em: http://localhost:5173
